Como calculamos o nível de risco

Fórmula v0 — agregamos, não inventamos

O nível de risco mostrado para o Maranhão (Perigo / vermelho) reflete diretamente o que as fontes oficiais estão dizendo. Como há um aviso ativo classificado como Grande Perigo (severidade high) pelo INMET, todo o estado é marcado como Perigo. Não criamos níveis próprios nem rebaixamos avisos oficiais.
